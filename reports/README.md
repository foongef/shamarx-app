# Reports

This directory contains the canonical performance analysis for the trading bot.

## Current files

| File | What it is |
|---|---|
| `ANALYSIS.md` | The main report — strategy overview, multi-pair multi-account matrix, portfolio recommendation |
| `wide-matrix.json` | Raw 40-cell results (2 pairs × 5 accounts × 4 periods). Generated by `scripts/run-wide-matrix.ts` |
| `portfolio-2k.json` | Single-account simulation: $2k AUM trading XAUUSD + EURUSD together. Generated by `scripts/run-shared-account.ts` |

## Reproducing

```bash
# Make sure execution-service is up
docker compose -f docker/docker-compose.yml up -d execution-service postgres redis

# Wide matrix (per-pair, per-account, per-period)
TZ=UTC pnpm exec ts-node -P tsconfig.build.json --transpile-only scripts/run-wide-matrix.ts

# Shared portfolio simulation
TZ=UTC pnpm exec ts-node -P tsconfig.build.json --transpile-only scripts/run-shared-account.ts
```

`TZ=UTC` is required — without it, host-vs-docker timezone drift produces different sweep detection.
